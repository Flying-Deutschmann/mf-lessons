## 6. Function Task

## Instructions:
- Create a function that takes an argument (like a name) and prints a greeting message.
- Call the function with different names to see the result.

## Example:
```
#include <stdio.h>

// Function definition
void greet(char name[]) {
    printf("Hello, %s!\n", name);
}

int main() {
    // Function call
    greet("Alice");
    greet("Bob");

    return 0;
}
```
